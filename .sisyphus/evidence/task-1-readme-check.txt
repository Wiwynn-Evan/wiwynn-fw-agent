# wiwynn-fw-agent

## Overview
`wiwynn-fw-agent` is a firmware integration agent built with OhMyOpenCode (OMO) that orchestrates custom skills and commands for automated firmware development workflows. This project demonstrates the extensible architecture of OhMyOpenCode by integrating domain-specific automation tasks.

## OhMyOpenCode Skills
This project leverages OhMyOpenCode skills to enable intelligent automation:
- **jira-deep-analysis**: Integrates with Jira for ticket analysis and status tracking
- **GitHub integration**: Manages repository interactions and API operations
- **Firmware toolchain**: Custom skills for firmware build and deployment automation

The `.opencode/` directory contains skills, commands, and agents that extend the core OMO platform with project-specific functionality.

## Getting Started
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your API credentials
3. Load custom skills from `.opencode/skills/` directory
4. Run commands via the orchestrator

For more information on OhMyOpenCode, visit the [official documentation](https://opencode.dev).
